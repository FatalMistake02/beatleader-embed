# BeatLeader Embed

A simple BeatLeader player stats widget that can be embedded using an `<iframe>`.  
Displays:
- Player Global Rank and Total PP 
- Latest Play with PP and stars if ranked  

---
## Usage

### Option 1. Embed using an `<iframe>`

```html
<iframe 
    src="https://fatalmistake02.github.io/beatleader-embed.html?player=playerID" 
    width="440" height="90" frameborder="0">
</iframe>
```
Replace playerID with BeatLeader player ID

---

### Option 2. Use index.html with a hardcoded player ID

Download
